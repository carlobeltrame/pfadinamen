Wie funktioniert das?

Diese Webseite generiert mit künstlicher Intelligenz auf Knopfdruck neue Pfadinamen.

Künstliche Intelligenz ist ein Begriff für Software, die menschliches Verhalten nachahmt. Zum Beispiel ist ein Computergegner bei einem Videogame eine künstliche Intelligenz.

Ein Unterbereich davon ist das sogenannte "machine learning". So nennen wir es wenn in der Software nicht explizite Verhaltensregeln einprogrammiert sind, sondern das Programm anhand von vielen Beispielen selber lernt, wie es sich verhalten soll.

Dieser Generator enthält eine machine learning Software namens char-rnn. Diese wurde mit einer Liste von 20'000 Pfadinamen, Cevinamen und Jublanamen trainiert. Das Programm hat so gelernt, wie Pfadinamen normalerweise aussehen.

Die Namen werden Buchstabe für Buchstabe generiert. Das Programm schaut immer die bisher generierten Buchstaben an, und berechnet dann, welche Buchstaben als nächstes wie gut hinpassen.

Wenn zum Beispiel bisher Tarant... generiert wurde, dann wird wohl als nächstes ein u besser passen als ein x oder ein P. Der Generator wählt immer zufällig aus allen Buchstaben die er kennt aus, welchen Buchstaben er als nächstes schreiben soll. Aber je passender ein Buchstabe ist, desto höher ist die Chance dass der Generator diesen Buchstaben auswählt.

Diesen Pfadinamen-Generator hat Carlo Beltrame / Cosinus aus der Pfadi Gloggi in Zürich programmiert.

Es war eine Idee die ich schon lange mal hatte, sobald ich den Artikel "The Unreasonable Effectiveness of Recurrent Neural Networks" von Andrej Karpathy gelesen habe. Gemäss meinen Tests sind ungefähr drei Viertel der ausgespuckten Pfadinamen neuartig (d.h. nicht in der Liste der Trainingsdaten vorhanden).

How does this work?

This website generates new scout names using artificial intelligence.

Artificial intelligence is software which imitates human behaviour. For example, an NPC in a video game is an artificial intelligence.

One discipline of artificial intelligence is the so-called "Machine Learning". That is when a program does not have explicit behaviour rules programmed in, but rather learns from many examples how to behave.

This generator contains machine learning software called char-rnn. It was trained using a list of 20000 scout names, JuBla names and YMCA names. This way the program has learned what scout names normally look like.

The scout names are generated letter by letter. The program looks at the already generated letters, and calculates which letters fit next and how well.

For example, if Tarant... has already been generated, a u will probably fit better as the next letter than an x or a P. The generator always randomly chooses the next letter it writes from all the letters it knows. But the better a letter fits, the higher it's change of getting picked by the generator.

This scout name generator was programmed by Carlo Beltrame / Cosinus from Gloggi in Zürich.

It was an idea I have had for a long time, ever since I read the article "The Unreasonable Effectiveness of Recurrent Neural Networks" by Andrej Karpathy. According to my tests, around three out of four of the generated scout names are completely new (i.e. not present in the training data).
